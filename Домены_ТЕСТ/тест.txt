Маршрутизатор имеет коллекцию внешних доменов в памяти, которые не могут быть посещены пользователем локальной сети. Существует правило, что если домен заблокирован, то блокируются и все его поддомены.Например,если домен adult.hb заблокирован, также заблокированы следующие хосты:

	images.adult.hb, ringo.adult.hb, video.ringo.adult.hb

Написать функцию на C#

	class Solution { public static int[] solution(string[] A, string[] B); }

это, учитывая непустой массив A из N хостов и B из M заблокированных доменов, возвращает последовательность, состоящую из L целых чисел, где каждое целое число представляет индекс хоста во входном массиве, который может быть посещен пользователем.

Например, дано:

A[0] = unlock.microvirus.md                  B[0] = microvirus.md
A[1] = visitwar.com                                 B[1] = visitwar.de
A[2] = visitwar.de                                    B[2] = piratebay.co.uk
A[3] =fruonline.co.uk                              B[3] = list.stolen.credit.card.us
A[4] =australia.open.com
A[5] =credit.card.us

функция должна возвращать массив [1, 3, 4, 5], как объяснено выше.

Допустим что:

	•N и M целые числа в диапазоне  [1..20,000];

	•L целое число в диапазоне [0..20,000];

	•каждый элемент массива представляет собой строку длиной [2..256];

	•каждый элемент коллекции B представляет собой строку длиной[2..256].

Элементы входных массивов изменять нельзя
